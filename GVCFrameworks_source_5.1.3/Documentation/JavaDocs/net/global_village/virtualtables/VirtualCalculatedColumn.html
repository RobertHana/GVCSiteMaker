newCalculationContext

public Map newCalculationContext()
Returns an context to be used when evaluating an expression. This contains a calculatingColumns key that points to a list of columns that are in the process of being evaluated. The calculatingColumns list is used to detect circular expressions. For OGNL, this also contains a classResolver key. Member access is controlled by WhiteListMemberAccess to prevent attacks via calculated columns.

Returns:
an OGNL context to be used when evaluating an expression.
