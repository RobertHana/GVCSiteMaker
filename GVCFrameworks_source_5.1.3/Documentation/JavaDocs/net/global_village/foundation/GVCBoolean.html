Represents a Boolean value as an object. GVCBoolean can translate between the internal Boolean state and a string representation of 'Y' for true and 'N' for false. GVCBoolean was created to support storing boolean values in a database as character data. To create an attribute in EOModeler that is mapped to this class, set these items in the Attribute Inspector:

  • External Type: CHARACTER, VARCHAR or equivalent
  • Internal Data Type: Custom
  • External Width: 1 (or greateer)
  • Class: GVCBoolean
  • Factory Method: booleanWithString:
  • Conversion Method: stringValue
  • Init Argument: NSString
To manually set up an EOAttribute, use this code: myBooleanAttribute.setValueClassName("GVCBoolean"); myBooleanAttribute.setFactoryMethodArgumentType(EOFactoryMethodArgumentIsNSString); myBooleanAttribute.setValueFactoryMethodName("booleanWithString"); myBooleanAttribute.setAdaptorValueConversionMethodName("stringValue");
