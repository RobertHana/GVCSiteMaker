Class MenuElement

java.lang.Object
  net.global_village.woextensions.dynamicmenu.MenuElement
Direct Known Subclasses:
ChildMenuElement, MainLevelMenuElement, SubMenuElement
public abstract class MenuElement
extends Object

Super class for menu elements. A MenuElement is one item in the DynamicMenu. A MenuElement is capeable of producing a representation of itself as javascript function calls.
