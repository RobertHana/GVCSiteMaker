Package net.global_village.woextensions.components

Interface Summary
Confirmable This goes with ConfirmAction.
 

Class Summary
AlternatingColorRowComponent This component when used in conjunction with an HTML table and a WORepetition will give the desired alternating background colors on rows.
BreadCrumb This component creates a bread crumb trail, which looks like this: Main > User Settings > Demographic To use this, pass an NSArray of Crumbs to this component's trail binding.
BreadCrumb.Crumb The Crumb class, used to pass in to a BreadCrumb component.
ConfirmAction A component that allows user to confirm or deny a specific action.
CSSComponent CSSComponent is a WOComponent which can be added to a page to generate the HTML for:
<link rel="stylesheet" href="some.css" type="text/css">
where the href can vary depending on the OS, browser, and browser version that the user is using.
DataGroup A DataGroup organizes a sorted group of similar Enterprise Objects and is able to split it into sub-subgroups to model a hierarchy of EOs.
DrillDownTable A component that displays information of an array of Enterprise Objects in a drill-down effect.
Log4JConfigurationPage A simplified version of ERXLog4JConfiguration component which configures and manages the log4j logging system.
MappedListPopUp This component is an alternative to WOPopUp.
WO522FileUpload This is a hack around a bug in WOFileUpload introduced in WO 5.2.2.
