isNull

public static boolean isNull(Object aValue)
Returns true if value is null or an instance of NSKeyValueCoding.Null. This is useful when validation properties as in some situations, value will be an instance of NSKeyValueCoding.Null rather than null. For example, the FrontBase adaptor calls validateValue:forKey: as part of its fetch processing during which time null values are represented as EONullValue

Parameters:
aValue - value to check for null
Returns:
true if value is null or an instance of NSKeyValueCoding.Null
