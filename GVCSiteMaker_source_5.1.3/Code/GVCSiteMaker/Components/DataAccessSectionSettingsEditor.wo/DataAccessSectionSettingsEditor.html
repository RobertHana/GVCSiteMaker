Visitors to this section will be allowed to perform the following actions on the table:
Browse/Search for Records Add Records Delete Records Import Records
You must grant permission to Add Records if you want to use the "Add Record" Form as the first thing visitors see in this Section.
You must grant permission to Browse/Search for Records if you want to use the "Search Record" Form as the first thing visitors see in this Section.
  When visitors first enter this section, the set of records found will be:
All records in the table
Records found by the "Default Search", defined below A default search must be defined if you choose this option. When Browse/Search for Records is unchecked, no records can be found in this section.     When visitors first enter this section, they should see: If you want to use the "Add Record" Form as the first thing visitors see in this Section then you must grant permission to Add Records (above). If you want to use the "Search Records" Form as the first thing visitors see in this Section then you must grant permission to Browse/Search for Records (above). If you want to use the "Import Records" Form as the first thing visitors see in this Section then you must grant permission to Import Records (above).
