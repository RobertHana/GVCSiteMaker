hasReferringRows

public boolean hasReferringRows()
Returns true if any row in another table refers to data in this row. This is the case if a lookup field in a row of another table uses a field in this row as the source of its lookup data. If this returns true, then this row cannot be deleted (prevented by Deny deletion rule in VirutalField).

Returns:
true if row in another table refers to data in this row.
