isStable

protected boolean isStable(com.webobjects.eocontrol.EOEditingContext editingContext,
                           File file)
Determines if the file is stable enough to be processed, or if it is being modified. This implementation looks at the time of last modification and file size, waits stabilityCheckInterval() seconds, and checks for any change. If you don't want a stability check, override this method to return true. If you want to wait for more or less time override stabilityCheckInterval().

Parameters:
editingContext - used to verify stability
file - the File to check for stability
Returns:
true if the lastModified date and file size remain the same for N seconds
See Also:
stabilityCheckInterval()
