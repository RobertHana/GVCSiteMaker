public class MainLevelMenuElement
extends MenuElement
implements com.webobjects.foundation.NSKeyValueCoding

A MainLevelMenuElement is an item in a DynamicMenu at the topmost level. Elements at this level have more capabilities and options then elements at the lower levels. These elements are also initially visible when the page is drawn.
