Class AbstractStringTokenizer

DelimitedStringTokenizer, FixedWidthStringTokenizer
public abstract class AbstractStringTokenizer
extends Object
implements Enumeration

Abstract class representing a tokeniser that accepts a string and breaks it into tokens. Each token can be associated with an optional name. You can use the enumeration interface to return the strings one by one or get them all at once using the allRemainingTokens() method.
