countOfObjectsForEntityNamed

public static Number countOfObjectsForEntityNamed(com.webobjects.eocontrol.EOEditingContext editingContext,
                                                  String entityName,
                                                  com.webobjects.eocontrol.EOQualifier qualifier)
Returns the number of objects of the specified entity, optionally restricted by qualifer. This should be used when fetching all the objects is not desirable and server side evaluation is needed.

Example Usage:

    EOQualfier unpaidInvoices = EOQualfier.qualifierWithQualifierFormat("isPaid = 'N'", null);
    Number numberOfUnpaidInvoices = countObjectsForEntityNamed(editingContext,
                                                               "Invoice",
                                                               unpaidInvoices);

Parameters:
editingContext - the editingContext to be used for fetching
entityName - the entity name of the objects to be counted
qualifier - the qualifier to be applied
Returns:
the number of objects of the specified entity

computeAggregateFunction

public static Object computeAggregateFunction(com.webobjects.eocontrol.EOEditingContext editingContext,
                                              String aggregateFunction,
                                              String attributeName,
                                              String entityName,
                                              com.webobjects.eocontrol.EOQualifier qualifier)
Computes aggregate function (min, max, sum, and avg are currently supported) on the attribute named attributeName on the entity named entityName, the objects operated on are optionally restricted by qualifer. The result is returned as the appropriate class. NOTE: there is a bug in the SQL generation facility that this method uses, so if you use a qualifier that qualifies based on a literal, you will have to properly quote and escape the literal before creating the qualifier (see the example). Qualifiers used elsewhere seem to work fine...

Example Usage:

    EOQualifier unpaidInvoices = EOQualfier.qualifierWithQualifierFormat("isPaid = 'N'", null);
    BigDecimal totalOfUnpaidInvoices = EOEditingContextAdditions.computeAggregateFunction(editingContext, "sum", "grandTotal", "Invoice", unpaidInvoices);

Parameters:
editingContext - the editing context to be used for fetching
aggregateFunction - the aggregate function to be applied to the attribute
attributeName - the name of the attibute to be evaluated
entityName - the name of the attribute's entity
qualifier - the qualifier to be applied to the fetch
Returns:
the result of the aggregate function, null is returned if there are no matching records in the database
