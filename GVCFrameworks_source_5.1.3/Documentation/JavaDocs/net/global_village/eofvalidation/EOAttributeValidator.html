Class EOAttributeValidator

java.lang.Object
  net.global_village.eofvalidation.EOAttributeValidator

public class EOAttributeValidator
extends Object

A collection of static methods which validate a value for an attribute. The only method you should ever need to call directory is displayNameForAttribute.

checkTypeCompatibility

public static void checkTypeCompatibility(Object value,
                                          com.webobjects.eoaccess.EOAttribute attribute)
                                   throws EOFValidationException
Raises a validation exception if value is not compatible with the internal type (class) that this attribute is mapped to. The value to is compatible in three cases:
  • it is of the same class as the value class for this attribute
  • it is of a sub-class of the value class for this attribute
  • it is null or EONullValue

isGeneratedPrimaryKey

public static boolean isGeneratedPrimaryKey(com.webobjects.eoaccess.EOAttribute attribute)
Returns true if this attribute represents a primary key which will be generated by EOF. This attribute is considered to represent a primary key which will be generated if:
  • it is part of the primary key
  • there is only one column in the primary key (EOF does not generate for compound keys)
  • the attribute type is appropriate for generation (NSNumber or 12 byte NSData)

Parameters:
attribute - EOAttribute to check
Returns:
true if the attribute is a generatable primary key
