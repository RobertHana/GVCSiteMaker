testConfigurationDictionary

protected com.webobjects.foundation.NSDictionary testConfigurationDictionary()
Returns the configuration dictionary for this test run. This class gets the dictionary from the TestConfigurationDictionary.plist of the framework in the working directory when the tests are run.

The dictionary is expected to have this format: { MailServer = "mail.someserver.com"; POPAccount = "you@someserver.com"; MailUser = "you"; MailUserPassword = "secret"; }

Returns:
the configuration dictionary for this test run
