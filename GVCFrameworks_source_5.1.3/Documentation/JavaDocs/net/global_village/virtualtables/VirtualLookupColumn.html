A lookup column is one that references a specific column in another table (actually, it could be the same table, but that wouldn't be very useful). The data in the referenced column could be used to populate a popup list.

Used by the table import process to perform additional processing on an imported field value. This subclass looks for the value in it's lookup table and returns the lookup value's FK.

Note that no coercion is done, so value must be the correct type.
