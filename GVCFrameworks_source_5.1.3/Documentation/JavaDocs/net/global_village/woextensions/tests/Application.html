public class Application
extends WOApplication

This class is added in this test package as a work-around for the tests to successfully run. Without this class, an error "Class 'Session' exists (class javax.mail.Session) but is not a subclass of WOSession", will be raised. The reason for this is that, when it tries to create a session for these tests, it looks for any class named Application.
