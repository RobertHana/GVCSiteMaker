replaceFormattingWithHTML

public static String replaceFormattingWithHTML(String originalString)

Replaces non-HTML formatting in originalString with HTML formatting. This is useful when displaying text that has been uploaded or cut and pasted without the benefit of added HTML formatting.

Note: at present the only change is that line breaks are replaced with the HTML_LINE_BREAK sequence.

Parameters:
originalString - the string to convert to HTML formatting
Returns:
originalString with non-HTML formatting replaced with HTML formatting
