public class SecondaryMenu
extends DynamicMenu

A subclass of DynamicMenu which represents menus which appear when MenuElements are "moused over". If isSubMenu() is true this menu is a SecondaryMenu of a MainLevelMenuElement if isChildMenu() is true this menu is a SecondaryMenu of a SubMenuElement. It is intended that SecondaryMenu only be used as a submenu or a child menu.
