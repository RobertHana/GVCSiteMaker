Class ChildMenuElement

java.lang.Object
  net.global_village.woextensions.dynamicmenu.MenuElement
      net.global_village.woextensions.dynamicmenu.ChildMenuElement

public class ChildMenuElement
extends MenuElement

A ChildMenuElement is an item in the DynamicMenu below (beside) a SubMenuElement. These elements are not visible when the menu is first drawn but only when a SubMenuElement is "moused over".
