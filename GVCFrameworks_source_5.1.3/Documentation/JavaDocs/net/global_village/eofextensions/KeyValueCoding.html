copyMappedValues

public static void copyMappedValues(com.webobjects.foundation.NSDictionary source,
                                    com.webobjects.foundation.NSDictionary mapping,
                                    com.webobjects.foundation.NSKeyValueCoding destination)
Copies values from source into destination based on mapping. Keys in mapping are used to extract values from source. The resulting value value is set into destination with a key that is the value associated with the key in mapping. Expressed in code: 
 destination.takeValueForKey(source.valueForKey(key), 
                             mapping.objectForKey(key));
Uses safeTakeValueForKey() to move the value into destination. Values in source that do not correspond to a key in mapping are ignored. This method is similar to EOKeyValueCodingAdditions.takeValuesFromDictionaryWithMapping but does not require a one to one mapping between source and mapping.

Parameters:
source - keyed dictionary of values to put into destination
mapping - keyed dictionary of names in destination
destination - object to receive values from source under the names given in mapping

takeValuesForKeys

public static void takeValuesForKeys(com.webobjects.foundation.NSKeyValueCoding source,
                                     com.webobjects.foundation.NSKeyValueCoding destination,
                                     com.webobjects.foundation.NSArray keys)
Moves values from source into destination based on the contents of keys. Essentially, this is a filtered version of EOKeyValueCodingAdditions.takeValuesFromDictionary where the contents of keys filters which values are moved. Uses safeTakeValueForKey() to move the value into destination.

Parameters:
source - the object where the values will come from
destination - the object where the values will go
keys - list of keys determining which values will be moved

safeTakeValueForKey

public static void safeTakeValueForKey(com.webobjects.foundation.NSKeyValueCoding destination,
                                       Object value,
                                       String key)
Implements takeValueForKey and safely handles both null and NSKeyValueCoding.NullValue values. If value is null and destination does not implement EOKeyValueCoding, NSKeyValueCoding.NullValue is set into destination. If value is NSKeyValueCoding.NullValue and destination implements EOKeyValueCoding, null is set into the destination.

Parameters:
destination - object to receive value
value - value to put into destination
key - name to put value under in destination
