Package net.global_village.woextensions.securecomponents

Class Summary
SecureActionComponent Super class for Secure Components that take in an Action binding
SecureBrowserComponent A Browser List Input control which can conditionally display itself as a Browser or a series of strings.
SecureButtonComponent A Form Button which can conditionally display itself as a button or not display at all.
SecureCheckBoxComponent If isPermissable is true displays a checkbox controlled by isChecked, if isPermissable is not true displays a string value for isChecked.
SecureComponent This is the super class for all Secure Components
SecureDropDownComponent A Drop Down List Input control which can conditionally display itself as a PopUp or a string.
SecureHyperlinkComponent A HyperLink which can conditionally display itself as a HyperLink or nothing.
SecureTextAreaComponent A Text Input control which can conditionally display itself as a TextArea or a string.
SecureTextFieldComponent A Text Input control which can conditionally display itself as a TextField or a string.
