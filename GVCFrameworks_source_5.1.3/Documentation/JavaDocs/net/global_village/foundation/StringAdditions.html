This will determine if an array of words is contained within the string. Either ensuring all words are matched, or at least 1 word is matched. Case does not matter. collectionOfWords - collection of strings to search for
ensureAllWordsMatched - true if all words in arrayOfWords should be matched
Returns:

escapeSQLForFrontBase

public static String escapeSQLForFrontBase(String sourceString)
Converts sourceString to a valid SQL for FrontBase. Currently, EOF has a known bug in Java: it does not properly quote string literals. When this method is called, it will add "'" to both sides of the string and add the FrontBase's escape character (which is "'") for all occurences of "'". eg. "Smith" => "'Smith'" "O'Brien" => "'O''Brien'"

Parameters:
sourceString - the string to convert
Returns:
the string properly quoted
