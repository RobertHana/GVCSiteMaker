Class SubMenuElement

java.lang.Object
  net.global_village.woextensions.dynamicmenu.MenuElement
      net.global_village.woextensions.dynamicmenu.SubMenuElement

public class SubMenuElement
extends MenuElement

A SubMenuElement is an item in a SecondaryMenu below the MainLevel. These elements are not visible when the menu is first drawn but only when a MainLevelMenuElement is "moused over".
