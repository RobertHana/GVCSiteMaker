public class AlternatingColorRowComponent
extends com.webobjects.appserver.WOComponent

This component when used in conjunction with an HTML table and a WORepetition will give the desired alternating background colors on rows. Use this WOComponent instead of the standard TR tag. Pass in the index of the WORepetition that envelopes this WOComponent.
