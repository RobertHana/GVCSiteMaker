er.extensions.migration.IERXMigration
public abstract class ComplexMigration
extends er.extensions.migration.ERXMigrationDatabase.Migration

Migration that provides for migrations that combine changes from Java code, SQL scripts, and DBUnit datasets. This assumes that any SQL files are in a specific structure under Resources: 

 Resources/
     <OptionalDirectory>/
         <DatabaseVendorName>/
             <scripts go here>/

runScriptForMigration

protected void runScriptForMigration(er.extensions.migration.ERXMigrationDatabase database,
                                     String scriptPath)
                              throws SQLException
Runs a database and migration specific script if it exists and throws if it does not. The script name is formed as <scriptPath>/<database.productName>/<migrationClass.simpleName>.sql". For example when migrating to version 6 of MyModel, calling this for FrontBase with a scriptPath of SQL the name would be "SQL/FrontBase/MyModel6.sql.

Parameters:
database - ERXMigrationDatabase that the migration is for
scriptPath - path to database specific script directories relative to the Resources directory
Throws:
SQLException - if the script fails at the database

importDataAndUpdatePKSequence

protected void importDataAndUpdatePKSequence(com.webobjects.eocontrol.EOEditingContext editingContext,
                                             er.extensions.migration.ERXMigrationDatabase database,
                                             String resourcePath)
                                      throws ClassNotFoundException,
                                             SQLException,
                                             org.dbunit.DatabaseUnitException
Inserts the contents of the indicated resource as DBUnit data set. This should be called only after the schema is complete as later ALTER TABLE or ALTER COLUMN statements will cause an SQL error due to the uncommitted data inserted by this method. The primary key sequence for each table data is imported into can be updated by the number of rows imported.
