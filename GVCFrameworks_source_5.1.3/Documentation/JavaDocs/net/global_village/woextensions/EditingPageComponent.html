EditingPageComponentForEOProperty
public abstract class EditingPageComponent
extends WOComponent

Base class for data input components that will be used on an EditingPage subclass. It provides a caption, description, concept of required, and validation error reporting. It also provides a unique identifier for this component and easy access to the editing context, page, and session.
