public class ScaledSumOperator

implements com.webobjects.foundation.NSArray.Operator

An operator class that computes the sum of an array's elements based on a keyPath. For example, to compute the sum of the hours of an array of WorkDetails, the method valueForKeyPath can be used with "@sumScale2.hours" as key path.The operatorNames method returns the keys for the operators that NSArray knows about, and operatorForKey returns the operator for a specified key. This class can be made available for use with NSArrays with the method setOperatorForKey. 

 // Register a summing the array operator that scales everything to two decimal places.  The name of the operator is @sumScale2
 new ScaledSumOperator(2);
