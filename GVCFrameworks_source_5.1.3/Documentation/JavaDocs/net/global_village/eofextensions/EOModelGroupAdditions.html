Parameters:

adaptorName - the adaptor that the connectionInfo is for.
connectionDictionary - the connection dictionary that will be used for connecting the models

fetchAllCachedEntities

public static void fetchAllCachedEntities(com.webobjects.eoaccess.EOModelGroup modelGroup,
                                          com.webobjects.foundation.NSArray modelsToPrefetch)
Looks in all named models for entities marked as "cache in memory" and fetches all objects in these tables so that the object store will be prepopulated. This allows this to be done at program startup during a known time frame and also ensures that these objects are in memory and can be referred to during awakeFromFetchInEditingContext without firing a fault requiring database access.

Parameters:
modelGroup - the model group to be used for fetching
modelsToPrefetch - the names of the models to be prefetched
