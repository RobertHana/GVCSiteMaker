Creates and configures a new JDBC database connection based on an EOModel.

Parameters:
model - provides JDBC URL and credentials and used with ec to locate the EOAdaptor
ec - used with model to locate the EOAdaptor
shouldQualifyTableNames - enables or disables multiple schemas support. If true, DBUnit accesses tables with names fully qualified by schema using this format: SCHEMA.TABLE. If false, the single schema name is set in the IDatabaseConnection. If this is enabled, XML datasets need to have the schema included, e.g. <table name="SCHEMA.TableName">
escapePattern - optional pattern to escape names in SQL, "\"?\"" is the standard double quote
Returns:
the created and configured database connection
Throws:
ClassNotFoundException - if the JDBC driver class can't be loaded
SQLException - if a connection can't be made
org.dbunit.DatabaseUnitException - if DBUnit can't connect
