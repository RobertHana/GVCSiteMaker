isIntegrityConstraintViolation

public static boolean isIntegrityConstraintViolation(com.webobjects.eoaccess.EOGeneralAdaptorException exception)
Returns true if this exception is from an integrity constraint violation. This is detected by checking for an X/Open SQL CAE SQLState of 23XXX in the nested SQLException. See linked pages for lists of the error codes.

The SQLException documentation also states that this can be an SQL 99 error state. SQL99 codes are not yet supported by this method.

Parameters:
exception - the exception to check.
Returns:
true if this exception is an integrity constraint violation
See Also:
SQLState 1, SQLState 2, SQLState 3
