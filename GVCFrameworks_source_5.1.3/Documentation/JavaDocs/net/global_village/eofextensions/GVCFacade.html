public abstract class GVCFacade
extends Object

GVCFacade is the base class for the EO Facades. A facade can be defined as a showy misrepresentation intended to conceal something unpleasant. And that is what GVCFacade is used for, it provides a consistant interface to it's EOEnterpriseObject controlling how the object is displayed and how it is modified.
